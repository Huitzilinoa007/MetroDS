# ISSSTEC 

Sistema de gestión hospitalaria desarrollado en C++ como proyecto final de Algoritmos y Estructuras de Datos.

## Funcionalidades

- Registro y administración de pacientes.
- Gestión de consultorios.
- Lista de espera con prioridades.
- Atención y finalización de consultas.
- Historial de acciones realizadas.
- Estadísticas de pacientes.
- Ordenamiento de registros mediante Merge Sort.

## Estructuras de Datos

- Listas simplemente enlazadas.
- Cola de prioridad.
- Pila.
- Recursividad.

## Tecnologías

- C++
- Estructuras dinámicas
- Merge Sort
- Búsqueda recursiva
